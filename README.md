## How to Install

#### Clone this repository

$ git clone https://github.com/nathandallas/fiber-art-grid.git

#### Go into the repository

$ cd fiber-art-grid

#### Install dependencies

$ npm install

#### Run the app

$ npm run dev

Runs the app in the development mode.
